This is a set of documents I maintain about the possible making of a https://www.raspberrypi.com/products/raspberry-pi-pico/[Raspberry Pi Pico] based self-contained computer reminiscing 80’s and 90’s home computers.

There are also some ideas about a heroic fantasy RPG that could be implemented on this.
